# Background & Logo Images for Smart Solar Cloud

Please place your images in this directory with the following filenames:

### 1. Login Background Image
- **Filename**: `bg_login_lekise.png` (or change the file extension in `/src/pages/Login.tsx` to match your image type, e.g., `.jpg` or `.webp`)
- **Path**: `/public/images/bg_login_lekise.png`

### 2. Login Logo Mascot Image
- **Filename**: `lekise-mascot.png`
- **Path**: `/public/images/lekise-mascot.png`

Vite will serve these images automatically both in development mode and production.
